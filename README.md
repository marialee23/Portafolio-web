# Portafolio-web
Portafolio web sobre el Uso Ético de la Inteligencia Artificial en el Aula.

## Descripción
Sitio web creado para estudiantes de séptimo semestre de la Licenciatura en Química de la Universidad Distrital Francisco José de Caldas. Incluye:
- Presentación de conceptos éticos en IA.
- Recursos interactivos y diagramas.
- Actividades prácticas y recomendaciones.

## Archivos
- `index.html` - página principal.
- `styles.css` - estilos personalizados con paleta cálida.
- `script.js` - comportamiento de navegación activo.

## Cómo usar
Abrir `index.html` en un navegador compatible o servir el archivo con un servidor local.

## Recursos usados
- Bootstrap 5 (MIT)
- Bootstrap Icons (MIT)
- Google Fonts `Inter` (Open Font License)
- Ilustraciones propias con SVG/CSS
